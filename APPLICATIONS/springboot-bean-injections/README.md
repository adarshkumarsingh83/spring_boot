# springboot-bean-injection-example

* http://localhost:8080/api/beans
```
{
  "apiBeanOne": {
    "name": "API Bean One"
  },
  "apiBeanTwo": {
    "name": "API Bean Two"
  },
  "apiBeanThree": {
    "name": "API Bean Three"
  }
}
```

* http://localhost:8080/api/transformers

```
{
  "requestTransformer": {
    "name": "RequestTransformer"
  },
  "responseTransformer": {
    "name": "ResponseTransformer"
  }
}
```

* http://localhost:8080/api/customApis
```
{
  "myCustomApiOne": {
    "name": "MyCustomApiOne"
  },
  "myCustomApiTwo": {
    "name": "MyCustomApiTwo"
  }
}
```

* http://localhost:8080/api/customApiBeans
```
{
  "customApiBeanOne": {
    "name": "Custom API Bean One"
  },
  "customApiBeanTwo": {
    "name": "Custom API Bean Two"
  },
  "customApiBeanThree": {
    "name": "Custom API Bean Three"
  }
}
```
